# Ethics Stress-Test Frontend

Next.js demo frontend for the EECS 6895 project.

## Run Locally

```bash
npm run dev
```

Open `http://localhost:3000`.

## Pages

- `/` — project overview
- `/dataset` — filter and browse all 180 unified scenarios
- `/dataset/[id]` — full scenario record and metadata
- `/traces` — curated demo traces
- `/traces/[id]` — full evaluation timeline
- `/scorecard` — aggregate scorecard with Proposer model selector
- `/models` — 30-scenario cross-model scoreboard

## Static Data

- `public/scenarios.json` — generated from `data/base_scenarios.jsonl`
- `public/traces/*.json` — selected demo traces from `runs/pilot30_csr_judge`
- `public/aggregate.json` — original single-run scorecard artifact
- `public/model_scoreboard.json` — generated by `scripts/12_generate_model_scoreboard.py`

## Build

```bash
npm run build
```

The app is static-export friendly. The build may need network access because `next/font` fetches Geist font metadata during compilation.
